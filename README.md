# Curso IA

Repositorio para construir modulos didacticos sobre conceptos de IA generativa.

La documentacion esta organizada como un sistema escalable:

- `doc/`: sistema comun y modulos de aprendizaje.
- `doc/modulos/skills/`: primer modulo, dedicado a crear skills para agentes.
- `skills/term-course-builder/`: skill reutilizable para crear nuevos modulos siguiendo el mismo metodo.
- `raw/`: fuentes originales y contexto de trabajo.

Empieza por `doc/README.md`.
